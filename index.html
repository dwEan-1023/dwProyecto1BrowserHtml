BROWSER Y HTML

BROWSER:

En el campo informático, cuando decimos “browser” nos referimos a los navegadores, que son, básicamente un software que permite el acceso a la web, de tal manera que nosotros como usuarios podamos visualizar e interactuar de manera fácil y cómoda diferente contenido.

La función más importante del navegador es tomar e interpretar documentos escritos en lenguaje HTML que se encuentren en un servidor www (Word Wide Web) o en un servidor local y después mostrarlos en pantalla.

Algunos de los navegadores web más conocidos son: Google Chrome, Mozilla Firefox, Internet Explorer, Safari, Opera, entre otros. Que, por lo general, permiten mostrar gráficos, animaciones, ejecutar audios, etc.

HTML:

HyperText Markup Language por sus siglas en inglés, y en español “lenguaje de marcas de hipertexto”, es un lenguaje que se utiliza para la elaboración de páginas web, se basa en un sistema de formato de código abierto y el texto se crea a partir de etiquetas, las cuales el navegador interpreta y luego lo muestra en pantalla.

Para crear una página web con HTML es necesario contar con un documento de texto con extensión .html que tenga escrito todas las etiquetas necesarias, para crear lo que queremos mostrar en pantalla, es importante comprender, que HTML no es un lenguaje de programación ya que no cuenta con variables y estructuras de control.
