CONSORCIO W3C

El W3C, es decir, el consorcio World Wide Web es una sociedad en donde cada compañía que sea considerada miembro, se encarga de realizar juntamente estándares para la Web para poder lograr la tan famosa “web unica”, es decir, que las especificaciones para los formatos y protocolos Web sean compatibles unas con otras y a su vez permitir que (cualquier) hardware y software puedan acceder a ellas. Para esto cuenta con foros para discutir necesidades actuales, protocolos y pautas; Algunos objetivos principales de dicho consorcio son:

1) Web para todos: Expandir y hacer más alcanzables los medios para que se encuentren más personas aprovechando la tecnología.
2) Web desde cualquier sitio: aumentar cada vez más los dispositivos por medio de lo que se puedan navegar por la red.
3) Confianza: Permitir que la tecnología tenga un nivel de confianza alto, para los usuarios.
