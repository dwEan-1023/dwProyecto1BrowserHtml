¿Que es HTML?

HyperText Markup Language por sus siglas en inglés, y en español “lenguaje de marcas de hipertexto”, es un lenguaje que se utiliza para la elaboración de páginas web, se basa en un sistema de formato de código abierto y el texto se crea a partir de etiquetas, las cuales el navegador interpreta y luego lo muestra en pantalla. Para crear una página web con HTML es necesario contar con un documento de texto con extensión .html que tenga escrito todas las etiquetas necesarias, para crear lo que queremos mostrar en pantalla, es importante comprender, que HTML no es un lenguaje de programación ya que no cuenta con variables y estructuras de control;
