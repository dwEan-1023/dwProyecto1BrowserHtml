HISTORIA DEL HTML

En 1989 el lenguaje de etiqueta que predominaba era SGML y este era un lenguaje muy básico, luego en 1991 el norteamericano Tim Berners-Lee propone crear HTML como subconjunto del SGML, y da a conocer por medio de una publicación, 22 etiquetas de este lenguaje, describiéndolas cada una de una manera muy sencilla, además mostraba algunos componentes.

Cabe recalcar que muchos de esos componentes hoy ya no se utilizan o existen otros que se han agregado a medida que pasa el tiempo. Actualmente están los llamados “Editores Web” que permiten mediante herramientas gráficas, diseñar páginas web sin tener que codificar o ni siquiera ver el código que se genera, este proceso es llamado WYSIWYG (What You See Is What You Get), es decir, “lo que ves es lo que obtienes”. Esta es una práctica muy poco recomendada si se quiere aprender o trabajar seriamente en el campo informático.
