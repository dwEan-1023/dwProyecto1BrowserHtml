ETIQUETAS HTML

Teniendo en cuenta que HTML es un lenguaje de marcas (etiquetas), son estas marcas las que definen la presentación de la página web que se quiera crear. Una etiqueta o marca, siempre se definirá entre los simbolos, mayor que y menos que < aqui va el nombre de la etiqueta >

Algunas etiquetas elementales son:

< html > Representa la raíz de un documento HTML.
< head > Define la coleccion de metadatos de enlaces, hojas de estilo, etc.
< title > Define el titulo de la página, este se muestra en la pestaña del navegador web.
< style > Etiqueta que se utiliza para proporcionar estilos a la página.
< body > Define el cuerpo del documento.
< b > Texto en negrita
< i > Texto en cursiva
< s > Texto en tachado
< table > Define una tabla
